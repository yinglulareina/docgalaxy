# Introduction to Machine Learning

Machine learning is a branch of artificial intelligence that gives systems the ability to learn and improve from experience without being explicitly programmed. Instead of writing rules by hand, we feed data to algorithms and let them discover patterns automatically.

## Core Paradigms

**Supervised learning** trains on labeled examples. The model learns a mapping from inputs to outputs (e.g., classifying emails as spam or not-spam based on past labels).

**Unsupervised learning** finds hidden structure in unlabeled data. Clustering algorithms group similar data points without being told what the groups should be.

**Reinforcement learning** trains agents by rewarding desired behavior and penalizing undesired behavior. The agent explores an environment and maximizes cumulative reward over time.

## Why It Matters

Traditional software requires a programmer to anticipate every scenario. Machine learning systems generalize from examples, making them powerful tools for perception tasks (vision, speech), prediction (stock prices, disease risk), and generation (text, images).

The field has accelerated dramatically since 2012 when deep neural networks achieved breakthrough results in image recognition, outperforming decades of hand-crafted feature engineering.
